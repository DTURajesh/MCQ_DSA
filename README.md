# MCQ_DSA
 - What will the output of the following code snippet?
```
   void solve() {
   int a[] = {1, 2, 3, 4, 5};
   int sum = 0;
   for(int i = 0; i < 5; i++) {
       if(i % 2 == 0) {
           sum += *(a + i);
       }
       else {
           sum -= *(a + i);
       }
   }
   cout << sum << endl;
  }
  
```
 A) 2\
 B) 15\
 C) Syntax Error\
 **D) 3**
